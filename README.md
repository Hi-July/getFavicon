# getFavicon
获取网站的Favicon图标并显示在你的网页上.
<br/>

### 演示
<a href="https://api.iowen.cn/favicon/" target="_blank">演示地址</a>
<br/>

### 安装使用
+ 上传到网站根目录
+ cache 文件夹给 755 权限
+ 然后访问 http://you.url/get.php?url=https://www.iowen.cn
<br/>

+ 如果出现获取不了的情况建议删除缓存再试一次
<br/>
